# Oui Button

![buttons](./screenshot.png)

```html
  <!-- Default button --->
  <button class="oui-btn">
    Default Button
  </button>

  <!-- Primary button --->
  <button class="oui-btn oui-btn__primary">
    Primary Button
  </button>

  <!-- Danger button --->
  <button class="oui-btn oui-btn__danger">
    Danger Button
  </button>

  <!-- Disabled button --->
  <button class="oui-btn" disabled>
    Disabled Button
  </button>
```
